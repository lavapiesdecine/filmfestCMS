filmfestCMS
===========

Gestor de contenidos para festivales de cine, desarrollado para la Muestra de Cine de Lavapies. http://www.lavapiesdecine.net

Para desarrollar este CMS hemos partido de varias tecnologías:

* HTML 5 / php 5.3 / mySQL
* jQuery / TinyMCE / fancybox
* Crystal Clear
* FPDF / php.mo
* OpenStreeMap OpenLayers / Google Maps


## Requisitos

* PHP 5.3 o superior
* MySQL 5.0 o superior
* Módulo mod_rewrite de Apache (requerido)
* Añadir permisos de escritura en
	- cms/core/config
	- cms/locale
	- cms/www/vista/skins
	- img
	- doc
	- install/locale
	

## Instalacion

* Descomprimir el proyecto 
* Ir a http://site.com o bien si se ha descomprimido en una carpeta http://site.com/carpeta 
* Rellenar los formularios
